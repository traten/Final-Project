Download the ZIP file and extract all files to the folder that your 7ZIP or your extraction program recommends (should be Final-Project-master). Do NOT rename your folder or any files within it. Check your folder to make sure it contains an empty folder called UserFiles. If it does not, create it.

Compile the program using the following command:
g++ -std=c++11 FinalProject.cpp -o FinalProject

Run the program using:
./FinalProject

You will now create a username and password, which can be used to login to take a quiz and view results.
Printing the movie list will return your latest quiz results. Logging out will not allow the user to access their previous results (for privacy purposes). Exiting the system will delete all users from the system. The Programmer's Top Picks returns a list of all our favorite movies from IMDB's Top 250!
